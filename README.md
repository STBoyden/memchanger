# Memory Changer

#### _A multi-platform memory editor for games and other software._

Run using `go`:

```bash
go run .
```

Or install to `GOPATH/bin`/`GOBIN`:

```bash
go install github.com/STBoyden/memchanger@latest
```

Then run with (assuming `GOPATH/bin`/`GOBIN` is in `PATH`):

```bash
memchanger
```
